The whole engine, one binary.

Born from one bad evening with a Python stack: download stalls, segfaults, missing deps, an OOM footgun. The layer with all the bugs is the layer that doesn't need Python — so mlx-bun replaces it. TypeScript on Bun, talking straight to MLX.

Bit-exact by oracle

Logit parity with mlx-lm is the correctness contract. Every model ships with tier-a token-identical parity tests against the Python reference.

OptiQ quants, first-class

Sensitivity-aware 4-bit quantization, mixed KV-cache quantization, hot-swap LoRA adapters on a quantized base — never reload the weights.

Every API you already use

OpenAI chat + Responses, Anthropic messages, SSE streaming, tool calling, vision. Point your SDK at localhost and go.

Agents, built in

mlx-bun pi drops you into a pi coding-agent session on this server — local model, local tools, zero cloud.
