Workflow · quantize

Quantize a model.

Shrink any supported model to 4 or 8-bit with OptiQ-grade group quantization — smaller on disk, faster to decode, and serveable the moment it's scanned. No Python in the path.

Source model

Point at a Hugging Face repo id or a local snapshot path. Inspect first to confirm support and size.

Anything on huggingface.co, or an absolute path to a local snapshot.

Configure

Lower bits = smaller and faster, with some quality cost. 4-bit is the balanced default.

4-bit · ~4× smaller than bf16 · the recommended default for local serving.
Weights per shared scale. 64 is standard; 32 trades a little size for slightly better fidelity.
Uniform applies the bits above to every layer. Mixed runs OptiQ's sensitivity sweep + knapsack to allocate per-layer precision toward a target average.
OptiQ measures each layer's sensitivity, then solves a knapsack to pick per-layer 4/8-bit hitting this average bpw (~2–5 min). Lower = smaller; higher = closer to bf16.
